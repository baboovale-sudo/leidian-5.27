# OLA UI v4 Light Minimal Patch

## Scope

This patch changes the WinForms visual styling only.

## Changes from v3

- Removed filled green, blue, and red button backgrounds.
- Buttons now use one neutral light-gray style.
- Important actions use subtle text color only:
  - Start / confirm: muted blue text.
  - Stop / close: muted red text.
- Reduced visual noise from button colors.
- Kept the enlarged v2/v3 layout scale.

## Not changed

- No business logic changes.
- No event binding changes.
- No task execution changes.
- No emulator/plugin logic changes.
